# MCP 简介

> 什么是 MCP？为什么它正在改变 AI 的游戏规则？

---

## 1.1 什么是 MCP？

**MCP（Model Context Protocol）** 是 Anthropic 在 2024 年推出的开放标准，专门用来**连接 AI 和外部世界**。

简单说：
> **MCP = AI 的 USB-C 接口**

以前：
- 每个 AI 工具都要单独开发一套对接方式
- 就像手机充电🔌 - 苹果用 Lightning，安卓用 Type-C，出门要带好几根线

现在：
- 统一标准，一次对接，处处可用
- 就像 USB-C - 一根线充所有

---

## 1.2 MCP 之前 vs MCP 之后

### ❌ 没有 MCP 时

```
AI: "我想帮你查一下今天的天气"
用户: "好啊"
AI: "抱歉，我查不到天气，因为我没法上网"
```

AI 就像被关在笼子里的小鸟，只能看，不能动。

### ✅ 有 MCP 后

```
AI: "让我帮你查一下天气"
(调用天气 MCP 服务器)
AI: "今天北京 15-25°C，晴天，适合出门~"
```

AI 有了手和眼睛，可以真正干活了！

| 能力 | 说明 | 示例 |
